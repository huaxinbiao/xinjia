新方法。 《Node.js开发指南》创建ejs项目的命令为： express -t ejs microblog。执行后，创建的是jade项目。 在express3.x，express4.x中创建ejs命令更新为： express -e microblog //即ejs，-j（即jade） 当然，最直接的，你也可以修改package.json里的定义来实现安装ejs。 PS：建立工程过程 1.必须得安装express框架把：express的安装命令也更新了，需要安装express-generator $ npm install -g express //全局安装 $ npm install -g express-generator //这段命令可别忘了，不然会提示“express命令找不到的” 2.建立网站工程结构： express -e ejs microblog 3.根据提示，进入文件夹安装 cd microblog //microblog为前面创建工程的文件夹 npm install 方法二 1、安装Nodejs 下载地址:https://nodejs.org/en/ 2、设置npm环境变量 默认情况 npm 全局包路径在这 C:\Users\计算名\AppData\Roaming\npm，显示我们是不想以后所有包都装在C盘；那就得重新设置包路径： npm config set prefix “D:\Program Files (x86)\nodejs\node_global” npm config set cache “D:\Program Files (x86)\nodejs\node_cache” 然后再重新在电脑的系统环境变量 path 加上 ;D:\Program Files (x86)\nodejs\node_global 这样，后面安装的npm包就直接在任意目录下敲相关命令了哈； 3、给npm安淘宝装镜cnpm 为啥要装呢，一个原因哈都懂的就是安装其他模块速度快！ npm install -g cnpm --registry=https://registry.npm.taobao.org 4、安装Express框架 npm install express -g npm install express-generator -g 5、新建一个项目 express -t ejs projectName 6、安装项目 //进入项目的目录 cd projectName //输入命令安装 npm install 7、运行 //注意,这个地方不是node app.js npm start 然后就可以在浏览器输入：http://localhost:3000 访问了
